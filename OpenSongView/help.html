Sets

Sets are groups of "items". Currently only Songs are supported.
Tap the button in the upper left corner or slide the whole main scren to the right. The second tab on the bottom shows your sets.

Adding Sets

A new set can be added by entering a name in the input field at the top of the Set tab.

Adding Songs to Sets

In a Set tap the "Add Songs" button at the top. Then you can select the songs you would like to add. Each tap adds the song again to the Set. So you can have one song multiple times in one set.

Removing Items

You can either remove an item by

  • swiping over an item
  • tapping the "Edit" button at the top
and tapping the "Delete" icon

Reordering Items

Tap the "Edit" button at the top and grab one of the handles that appeared next to each item. Drag the item to the desired position.
